Compatibility with other Asset Store packages:

2D Toolkit:
	If you are having issues with crashes at startup, you must enable the Skip: Fields With SerializeField Attribute option.
	If that doesn't help, then add (without the quotes) "tk2d*" to Skip Type.
	
NGUI:
	If you are having issues with crashes at startup, you must enable the Skip: Fields With SerializeField Attribute option.
	If that doesn't help, then add (without the quotes) "UI*" to Skip Type.
	
KGFMinimap:
	If you are having issues with crashes at startup, you must enable the Skip: Fields With SerializeField Attribute option.
	If that doesn't help, then add (without the quotes) "KGF*" to Skip Type.
